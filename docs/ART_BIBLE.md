# 場景立繪・美術聖經（World & Style Bible）

> 目的：讓所有地點立繪維持**同一個世界、同一種畫風、同一個年代**。
> 任何一張新場景，都必須先讀本檔，再依「全域規範」生成；**不要讓單張圖各自發揮**，否則會風格／年代不一致。

---

## 0. 一句話定位
**現代亞洲都會、紅燈區底色**的成人視覺小說場景背景；畫風對齊角色立繪 `keyu_*.png`（半寫實 anime / 韓系 manhwa 厚塗），**非像素風**（像素風只用在導覽用的城鎮小地圖 `townmap.png`，屬另一圖層）。

## 1. 世界觀（World）
- **年代**：現代，約當 21 世紀當代。有霓虹招牌、日光燈、鐵皮、電線、現代成衣與雜貨；**不要**古裝、民國旗袍、蒸汽龐克或科幻未來感。
- **地理**：一座被青山與河流環抱、仍留著**舊城牆**的虛構亞洲小城。新舊混雜、略帶潮濕頹敗，入夜後紅燈與招牌亮起。
- **五個區**（與遊戲資料一致，各有色溫與氣味）：
  | 區 | 定位 | 視覺基調 |
  |---|---|---|
  | 中區・商業 | 商店街、招牌密集 | 暖琥珀燈光、老式日光燈、貨架雜物 |
  | 東區・風月 | 娼院、紅燈 | 桃紅／品紅霓虹、脂粉、暈光 |
  | 北區・機構 | 警局、醫院、官府 | 冷藍灰、日光燈、磁磚、秩序感 |
  | 南區・陋巷 | 棚屋、公廁、暗巷 | 泥褐、昏黃、鐵皮鏽蝕、潮濕 |
  | 西區・住宅 | 紅瓦舊樓、住家 | 紅瓦暖綠、生活感、安靜 |
- **主角**：柯妤潔（成年女性，現代衣著）。場景立繪本身**不畫清晰人物**（避免與角色立繪畫風打架）；最多遠處模糊剪影。

## 2. 全域視覺規範（每張都要遵守）
- **媒材／筆觸**：數位厚塗 + 清晰線稿，半寫實，精緻度對齊角色立繪。參考感覺＝「成人視覺小說 CG 背景 / 韓系 manhwa 場景」。
- **比例與輸出**：**9:16 直幅，1080 × 1920 px，PNG**。滿版出血、不需去背。
- **鏡頭**：第一人稱「即將踏入」的代入視角——站在門口往裡望，或在街角望向建物。中景到廣角，平視或略低視角。
- **構圖**：主體偏上／中，**畫面最下方 1/3 留給較單純的區域**（會疊上半透明黑漸層＋地點名浮層，重要細節別壓在最底）。
- **光線**：戲劇性場景光，依區色溫定調（見上表）。善用光暈、反光、地面濕反光。
- **時間／天氣**：預設**入夜或黃昏**（最能撐紅燈區氛圍與沈浸感）；全系列統一，避免有的大晴天有的深夜。
- **文字**：招牌可留白或用**繁體中文**；不要日文假名、不要英文霓虹為主、不要亂碼。
